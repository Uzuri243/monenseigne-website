# MON ENSEIGNE - Site Vitrine

Site vitrine professionnel pour MON ENSEIGNE, entreprise congolaise spécialisée en signalétique et supports de communication.

## 🎯 À propos

MON ENSEIGNE crée des enseignes lumineuses, totems, façades et supports de communication sur mesure pour les entreprises en Afrique centrale.

**Villes couvertes :**
- Brazzaville
- Pointe-Noire
- Kinshasa
- Lubumbashi

## 📦 Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)

## 🚀 Déploiement

Le site est automatiquement déployé sur Vercel à chaque push vers la branche `main`.

**URL en direct :** https://mon-enseigne-site-1.vercel.app

## 📝 Mettre à jour le site

1. Modifiez les fichiers localement
2. Commitez les changements : `git add . && git commit -m "Description du changement"`
3. Poussez vers GitHub : `git push origin main`
4. Vercel déploiera automatiquement la nouvelle version
